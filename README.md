#### Build

``` bash
# 安装依赖
yarn

# 启动
yarn dev

# 打包
yarn build

```

---

### Feature
- 基础的文件上传、下载、删除、重命名、查看功能
- 按名称、日期、类型、大小排序
- 批量删除、新建和上传
- 拖曳操作
- 复制链接
- 查看文件响应头
- 多选删除上传
- 上传下载展示，以及历史记录
- 账号历史
- 右键菜单
- 快捷键操作
- 前进，后退功能
