#### Build

``` bash
# 安装依赖
yarn

# 启动
yarn dev

# 打包
yarn build

```

---

### 功能
- 基础的文件上传、下载、删除、重命名功能
- 多选删除上传
- 文件夹删除、新建和上传
- 拖曳上传
- 进度条
- 保存授权历史
- ctrl shift 方向键选择文件，alt 加方向键前进后退
